ID e CLASS

ID

O ID funciona como um CPF . Assim como só posso ter um numero de CPF, cada elemento só pode ter um valor de ID. O CPF é o único ou seja,ninguem pod ter um CPF igual ao meu. Isso significa que o ID também é o único então não é possiveol repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente ao ID em que você só pode ter um valor ,no CLASS você pode ter vários valor. Tambem diferente do ID,em que não se pode repetir um valor ,no CLASS você pode utilizar o mesmo valor para vários elementos.
